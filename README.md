## Setup

git clone https://github.com/Joanna-Chadzynska/graphql-app.git

_server setup_

- cd server from graphql-app folder
- npm install

_client setup_

- cd client from graphql-app folder
- npm install

_run app - development_

- cd server
- npm run dev
